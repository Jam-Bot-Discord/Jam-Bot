Jam-Bot

Commands

Here's a list of all the commands the bot supports

Moderation

  • kick - Kicks a specified user, usage: !kick @user REASON
  • ban - Bans a specified user, usage: !ban @user REASON
  • lookup - Displays information about a specifed user or role, usage: !lookup @user
  • purge - Bulk deletes a certain number of messages, usage: !purge 50
  • settings - Allows admins to configure settings, usage: !settings

General Commands

  • suggest - Allows users to suggest things, usage: !suggest Create a memes channel
  • owner - Displays the owner of the bot, usage: !owner
  • uptime - Displays the current uptime of the bot, usage: !uptime
  • ping - Displays the current latency of the bot, usage: !ping
  • help - Displays help text for a certain command, usage: !help ban
  • shutdown - Shutdowns the bot, usage: !shutdown - BOT OWNER ONLY
  • shorten - Shortens a url, usage: !shorten https://google.com

Image Commands

  • cat - Purrrrr, usage: !cat
  • dog - Woof, usage: !dog
  • fox - Fox, usage: !fox
  • image - Google image search, usage: !image cute dogs - NOTE: Literally searches the web, use at your own risk
  • stock - Stock image search, usage: !stock dogs - NOTE: Better quality images, however doesn't cover everything. Images from Pexels

Settings

  • Mod Log

    • channel - Sets the modlog channel, usage: !settings modlog channel #ModLog
    • logdeletes - Whether the bot should log deletes to the modlog channel, usage: !settings modlog logdeletes on|off
  • prefix - Sets the bots prefix, usage: !settings prefix $
  • suggestions - Sets the suggestions channel, usage: !settings suggestions #suggestions
