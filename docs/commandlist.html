Command list

Here's a list of all the commands the bot supports.

General commands

Command Description Usage
Suggest Suggests something !suggest Make a memes channel
Owner Displays the bot owner !owner
Uptime Displays the bot's uptime !uptime
Ping Displays latency information !ping
Help Gets help on a specific command !help ban
Shorten Shortens a url !shorten https://google.com
Add Emoji Adds an emoji to the server !addemoji blobWave
Cactus Ouch !cactus
Tell Tells someone, something !tell @dom You are bad
Poll Makes a simple yes or no poll !poll Are chips good?

Moderation

Command Description Usage
Kick Kicks a specified user !kick @honk Being a noob
Ban Bans a specified user !ban @elle Not sleeping
Lookup Displays information about a specific user or role !lookup @honk
Purge Bulk deletes a specified number of messages in a channel !purge 50
Settings Configures the bot's settings !settings modlog
